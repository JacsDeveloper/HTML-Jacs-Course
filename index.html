ETIQUETAS GENERALES

br: Salto de línea, como dar un Enter (NO se cierra).
hr: Línea horizontal (NO se cierra).
b: Negrita.
i: Itálica - cursiva.
u: Subrayado
strike: Tachado.

div: Son contenedores, cajas invisibles de tipo bloque que pueden contener otros elementos. Su real uso se aprecia más en CSS.
span: Son contenedores, cajas invisibles de tipo inline. Su real uso se aprecia más en CSS.

a: Enlace/Link/hipervinculo.
El Atributo href es donde ponemos la ruta local o url del enlace.
El Atributo target="_BLANK" abre el enlace en una nueva pestaña.

p:

Está dispuesta para los parrafos. Lorem ipsum, dolor sit amet consectetur adipisicing elit. Placeat eligendi aperiam magni accusantium quia quibusdam corrupti totam recusandae accusamus iusto! Magni accusamus ipsa molestiae esse laboriosam neque repellendus, molestias quibusdam.


ol: Ordened List (Listas ordenadas). Items numerados, cada ítem va dentro de la Etiqueta li.
  1. Item 1
  2. Item 2
  3. Item 3

ul: Unordened List (Listas desordenadas). Items NO numerados (con viñeta), cada ítem va dentro de la Etiqueta li.
  • Item 1
  • Item 2
  • Item 3

select: Caja de selección de opciones/items. Dentro de select va la etiqueta option cada option es un item (es como un li).

ETIQUETAS DE MULTIMEDIA

img:

Insertar imagen. (No se cierra)
El Atributo src es para la ruta local o url de la imagen. (.jpg, .png, etc...)
El Atributo alt es para el texto descriptivo de la imagen en caso de que la imagen no cargue (Por ej: cuando no hay internet).
El Atributo title es para el título de la imagen y lo muestra al pasar el cursor sobre la imagen.
También se puede estipular el tamaño de la imagen con el Atributo width, sin embargo, todos los atributos referentes al tamaño, color, etc, se maneja a través de CSS.




video:

Insertar video.
El Atributo src es para la ruta local o url del video. (.mp4, .avi, .mov, etc.)
El Atributo alt es para el texto descriptivo del video en caso de que no cargue (Por ej: cuando no hay internet).
El Atributo title es para el título del video y lo muestra al pasar el cursor sobre la imagen.
El Atributo controls es para que el navegador nos preste los controles de reproducción (play/pause, fullscreen, volume, etc. Siempre hay que ponerlo).
También se puede estipular el tamaño del video con el Atributo width, sin embargo, todos los atributos referentes al tamaño, color, etc, se maneja a través de CSS.




audio:

Insertar audio.
El Atributo src es para la ruta local o url del audio. (.mp3, .ogg, .wav, .flac, etc.)
El Atributo alt es para el texto descriptivo del audio en caso de que no cargue (Por ej: cuando no hay internet).
El Atributo title es para el título del audio y lo muestra al pasar el cursor sobre el audio.
El Atributo controls es para que el navegador nos preste los controles de reproducción (play/pause, fullscreen, volume, etc. Siempre hay que ponerlo).




ETIQUETAS DE FORMULARIO

form:

Etiqueta que indica inicio del formulario. Los formularios son para captar datos, ingresarlos. Para ello se requiere definir el tipo de dato a ingresar, esto se hace a través de la etiqueta input.

input

Va dentro de la etiqueta form.
El Atributo type define el tipo de dato a ingresar.
El Atributo required hace OBLIGATORIO llenar ese campo, de lo contrario, al darle al botón Enviar/Submit, va a saltar un aviso que ese campo específico es OBLIGATORIO llenarlo.

  • <input type="text"/>

    El Atributo placeholder es la marca de agua que se muestra dentro de la caja de texto.
    El Atributo value es un valor/texto que se le da por defecto a la caja de texto, es igual a haberlo escrito.

  • <input type="password"/>

  • <input type="number"/>

  • <input type="email"/>

  • <input type="color"/>

  • <input type="date"/>

  • <input type="time"/>

  • <input type="range" min="1" max="6"/>

  • <input type="button" value="Texto dentro del Botón"/>

  • <textarea> <textarea/>

    Cuadro/Area de texto. Si tiene el Atributo readonly queda solamente lectura, inhabilitado para escribir.

  • <input type="submit"/>

    Envía los datos recogidos en las cajas anteriores.

ICONO

El ícono que sale al lado del título, en la pestaña del navegador, se escribe así:
<link rel="icon" href="/html icon.ico">

La etiqueta link con el atributo rel="icon" indican que se va a incrustar un ícono en el título, el atributo href es para la dirección local o URL del ícono (Por Ejemplo insertar icono de FontAwasome).
